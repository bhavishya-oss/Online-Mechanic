# Online-Mechanic
Java Web Application with spring mvc
This is a Java-based web application developed using NetBeans.
The project uses Java for backend logic and HTML/CSS for frontend design.


Technologies Used -
- Java
- HTML
- CSS
- Apache Topcat
- Netbeans IDE
  
Project Structure
- src/       → Java source code
- web/       → HTML, CSS, JSP files
- nbproject/ → NetBeans configuration
- build.xml  → Build configuration

Execution
The backend is executed locally using Apache Tomcat.
Source code is hosted on GitHub for version control and collaboration.
